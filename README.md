# Code Jam November 2025: Spending personality test

### Overview

The Spending Personality Test is a fun and engaging tool designed to help individuals understand their relationship with money. Through a series of carefully crafted questions, users receive personalized results that categorize their spending style and provide actionable insights for better financial decision-making.

## Description

Interactive Quiz Interface - Smooth, user-friendly question flow
Multiple Personality Types - Categorizes users into distinct spending personalities
Personalized Results - Detailed breakdown of spending habits and tendencies
Responsive Design - Works seamlessly on desktop, tablet, and mobile devices
Lightweight & Fast - Minimal load times with clean, optimized code
